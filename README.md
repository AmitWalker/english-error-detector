# 🔍 English Error Detector Pro

A professional desktop application for detecting and correcting errors in English sentences.

![Python Version](https://img.shields.io/badge/python-3.6%2B-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Linux-lightgrey.svg)

## ✨ Features

- **🔤 Spelling Check**: Detects 200+ common misspellings
- **📚 Grammar Analysis**: Identifies subject-verb agreement and more
- **✏️ Punctuation Correction**: Fixes punctuation errors
- **🎨 Style Improvement**: Detects informal language and wordiness
- **🤔 Confused Words**: Highlights commonly confused words
- **📊 Real-time Statistics**: Shows error counts by type and severity
- **🌙 Dark Theme**: Professional dark GUI
